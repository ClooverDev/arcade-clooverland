SUPER PLATFORM: REORIGINAL
These are the list from Super Platform: Reoriginal's updates, from V1.3.0 and beyond.

PORTUGUESE TRANSLATION
You can now choose between PT-BR and EN-US when starting the game, you can deactivate the language selection on session start through a option found on the Menu.
Portuguese Translation affects endings and any hud that has sprite, such as Level, Warning Screen, Menu Options, etc.

New GameJolt API's Code
- Moved the Endings "Trophies" on the Menu to the bottom of the logo. The little detail there is that now the icons that are on the left of the "R" are the first 6 ending on the game, and the ones on the right are from the Hunt or Hunted update.
- Replaced the old GameJolt API's code with a better one, which is a more recent code and that has better contact with the API itself, preventing crashes.

Other
- Optimized endings size and moved to WEBM file types, for lighter file size.
- Fixed some problems found throughout the process.

NEW ENDINGS
- Forest Ending
- Faker Ending
- Only the Faker Ending
- Stealer Endings (Pedro / Welly variants) 10% of replacing a ending
- Only One Ending

FAKER ARRIVED TO THE WOODS!
- Faker pursuit sequence added to the forest

New Warp Zone
- Unlock the Warp Zone's path after reaching any ending once, unlock the door itself to be usable after reaching the good ending once

Other
- Made some of the first songs louder (The Levels, Level 5/6, The Forest)
- New soundeffects for landing, walking, sprinting and for the beast
- Added a ending for the Miacrim on the credits screen
- Door transition
- Some bug fixes
